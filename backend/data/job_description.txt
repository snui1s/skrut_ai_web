Job Summary
พัฒนาและประยุกต์ใช้โมเดล AI (โดยเฉพาะ LLMs) เพื่อสร้าง Solution ที่ตอบโจทย์ธุรกิจ เน้นการทำ RAG และการเชื่อมต่อ AI เข้ากับ Software Ecosystem ของบริษัท

Key Responsibilities
AI Development: ออกแบบและพัฒนา AI Application โดยเน้นเทคนิค RAG และ Agentic AI

System Integration: เชื่อมต่อโมเดลผ่าน API และพัฒนา Backend รองรับ (FastAPI/Node.js)

Data Handling: จัดการข้อมูลและใช้งาน Vector Databases เพื่อเพิ่มประสิทธิภาพการค้นหาข้อมูล

Technical Requirements
Stack: Python, LangChain/LlamaIndex, Docker

AI Tools: มีประสบการณ์ใช้งาน OpenAI API, Anthropic หรือ Open-source LLMs

Database: เข้าใจการใช้งาน SQL และ Vector Search (เช่น Chroma, Pinecone หรือ Qdrant)

Mindset: เน้นการแก้ปัญหา (Problem Solving) และสามารถนำ AI Tools มาช่วยเพิ่มความเร็วในการ Dev ได้

Qualifications
วุฒิปริญญาตรี Computer Science หรือสาขาที่เกี่ยวข้อง

มี Logic ที่แม่นยำ และมีความเข้าใจใน AI Lifecycle
